  • Description

    千龙网·中国首都网是中共北京市委宣传部主管主办，由北京日报社、北京人民广播电台、北京电视台、北京青年报社、北京晨报社等京城主要传媒共同发起和创办的国内第一家综合性新闻网站，是北京网络舆论宣传主阵地。千龙网于2000年5月25日正式开通。2012年，千龙网·中国首都网在过去12年的发展基础上，着眼发展新趋势和舆论传播格局新变革，开拓创新，迈入新一轮发展阶段。
